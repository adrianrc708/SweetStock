\# SweetStock - Sistema de Gestión



Proyecto full stack en desarrollo (Spring Boot + React).



\## ¿Cómo ejecutar el proyecto?



\### Backend (Spring Boot)

1\. Crear `src/main/resources/application.properties` basado en `application-example.properties`

2\. Configurar credenciales de MySQL

3\. Ejecutar:



En resumen: Antes de ejecutar el backend, copiar application-example.properties como application.properties y agregar tus propias credenciales.


Luego abrir la terminal y ubicarse en la ruta en la que se encuentra el backend y ejecutarlo con 'mvn spring-boot:run'.
Después abrir otra terminal y ubicarse en la ruta en la que se encuentra el frontend y ejecutarlo con 'npm run dev'.
